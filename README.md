# 📝 Text Summarization Tool

This project is a simple desktop application built with Python and Tkinter that helps summarize long pieces of text into shorter, meaningful summaries. It uses a basic NLP-based extractive summarization approach by identifying the most important sentences based on word frequency.

## Features

* Simple and easy-to-use graphical interface
* Generates summaries from long text
* Adjustable summary length
* Copy summary to clipboard
* Displays basic summary statistics

## Technologies Used

* Python
* Tkinter
* Regular Expressions (re)

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python text_summarizer_gui.py
```
